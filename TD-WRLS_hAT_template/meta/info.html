This project template is the basis of an expansion board for the TD-WRLS wireless development kit.

This base project includes a PCB edge defined as the same size as the TD-WRLS PCB with the connectors placed correctly to align the two boards.

All IO present on the TD-WRLS board is connected to the project through the expansion headers.
